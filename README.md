# TELE-XKCD-BOT

Functional telegram bot for convenient reading xkcd comics.

## Features
- Notification for a new comic.
- Russian translation of comic (if it exists).
- Bookmarking.
- Searching by number and word.
 
Hidden features for admin:
- Antiflood.
- Logging.
- Broadcasting.
- Getting logs.


## Tech Stack

- aiogram
- aiosqlite
- loguru
- bs4 and requests
